# ClassLoaderCalculator

Wildfly 10

Run server :

```bash
   ./standalone.sh
```
or

```bash
   %JBOSS_HOME%\bin\standalone.bat
```

Build : 
```bash
   mvn clean install package
```

Deploy :
```bash
   mvn wildfly:deploy
```

Undeploy :
```bash
   mvn wildfly:undeploy
```

Redeploy :
```bash
   mvn wildfly:redeploy
```
